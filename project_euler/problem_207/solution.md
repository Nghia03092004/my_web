# Problem 207: Integer Partition Equations

## Problem Statement

For the equation $4^t = 2^t + k$, if we set $x = 2^t$, then $k = x(x-1)$. For each integer $x \geq 2$, this gives a distinct positive integer $k$ and a corresponding real $t = \log_2 x$. Each such $(k, t)$ pair is called a *partition* of $k$.

A partition is called *perfect* if $t$ is a positive integer, that is, if $x = 2^m$ for some $m \geq 1$.

Define $P(k)$ as the ratio of perfect partitions to total partitions among all valid $k$-values up to $k$. Find the least $k$ such that $P(k) < 1/12345$.

## Mathematical Development

### Substitution

Let $x = 2^t$. Then $4^t = (2^t)^2 = x^2$, so
$$
x^2 = x + k \implies k = x(x-1).
$$

For integer $x \geq 2$, the admissible values of $k$ are therefore
$$
2, 6, 12, 20, 30, 42, 56, 72, 90, 110, \ldots
$$

### Counting Partitions

At the boundary value $k = x(x-1)$, the total number of partitions counted by $P(k)$ is exactly the number of integers $y \geq 2$ with $y(y-1) \leq x(x-1)$, namely $x - 1$.

### Perfect Partitions

A partition is perfect precisely when $x$ is a power of 2. Thus the perfect values are generated by
$$
x = 2, 4, 8, 16, \ldots
$$
and at the boundary $k = x(x-1)$ the number of perfect partitions is
$$
\lfloor \log_2 x \rfloor.
$$

### Ratio Analysis

Hence, for $k = x(x-1)$,
$$
P(k) = \frac{\lfloor \log_2 x \rfloor}{x - 1}.
$$

As $k$ moves between two consecutive admissible values, neither the numerator nor the denominator changes, so the minimum is attained at one of these boundary points.

### Finding the Threshold

We need
$$
\frac{\lfloor \log_2 x \rfloor}{x - 1} < \frac{1}{12345},
$$
which is equivalent to
$$
x - 1 > 12345 \cdot \lfloor \log_2 x \rfloor.
$$

For $\lfloor \log_2 x \rfloor = 17$, we require
$$
x - 1 > 12345 \cdot 17 = 209865,
$$
so the first candidate is $x = 209867$. This indeed satisfies $\lfloor \log_2 209867 \rfloor = 17$, giving
$$
k = 209867 \cdot 209866 = 44043947822.
$$

## Editorial

The substitution $x = 2^t$ completely linearizes the problem. Instead of reasoning about real exponents, we only need to study the integer sequence $k = x(x-1)$. Up to the boundary value indexed by $x$, there are exactly $x-1$ admissible partitions, and the perfect ones are exactly those with $x$ equal to a power of 2, so there are $\lfloor \log_2 x \rfloor$ of them.

That means the ratio $P(k)$ at a boundary point is just $\lfloor \log_2 x \rfloor / (x-1)$. The answer is therefore the first $x$ for which $(x-1)$ overtakes $12345 \lfloor \log_2 x \rfloor$, and then we convert back to $k = x(x-1)$.

## Pseudocode

```text
target = 12345
x = 2

Repeat:
    perfect_count = floor(log_2(x))
    If x - 1 > target * perfect_count:
        return x * (x - 1)
    Increase x by 1
```

## Complexity Analysis

- **Time:** $O(x^\*)$, where $x^\*$ is the first integer satisfying the threshold inequality.
- **Space:** $O(1)$.

## Answer

$$\boxed{44043947822}$$
